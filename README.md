# Network Reloader
Network Reloader is a module that lets you use your events **almost the same way you did before adding it** and reduce their size by **a lot**.
I know that networking systems exist, but if you use this module, you won't have to change much and you will have control over event instances.

# Support
This module supports bindable and remote events.
Bindable and remote function support is being implemented.

# Installation
You can use either the Roblox model or the release file. You will need to import the release file into your game.
